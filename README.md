# Kolmogorov-Lipschitz Algorithm

Implementación del algoritmo de Kolmogorov-Lipschitz.

## Instalación

Este proyecto usa [uv](https://docs.astral.sh/uv/) para gestión de dependencias.

### Prerrequisitos
- Python 3.8+
- uv instalado

### Configuración
```bash
# Clonar repositorio
git clone https://github.com/redlab-math/kolmogorov-lipschitz-algorithm.git
cd kolmogorov-lipschitz-algorithm

# Instalar dependencias
uv sync
